# spybot-rs

A small CLI for the [Spybot](https://github.com/SmoFlaDru/Spybot2) Teamspeak server monitoring and statistics system. To fetch the current Teamspeak channels and clients.

![](screenshot.png)

## Requirements

- Rustup toolchain

## Install

1. Clone the repository
2. Run cargo install

```bash
$ cargo install --path .
```
